# 테스트 스크립트 안내

이 폴더에는 프로젝트의 각 기능을 검증하기 위한 테스트 스크립트가 포함되어 있습니다.

## 파일 목록

### test_yesterday.py — 어제 날짜 파이프라인 테스트
- **용도**: 어제 날짜의 영상을 대상으로 전체 파이프라인을 테스트합니다.
- **범위**: 수집 → 자막 추출 → Gemini 요약 → JSON 저장 → HTML 빌드
- **제외**: 카카오톡 전송, Git 배포 (실 서비스에 영향 없이 테스트)
- **실행**:
  ```bash
  python tests/test_yesterday.py
  ```
- **확인**: 실행 후 `docs/index.html`을 브라우저로 열어 결과 확인

### test_refresh.py — 카카오톡 토큰 갱신 테스트
- **용도**: 카카오톡 액세스 토큰이 만료되었을 때, 리프레시 토큰으로 갱신이 정상 작동하는지 확인합니다.
- **범위**: 토큰 파일 로드 → 갱신 API 호출 → 결과 확인
- **실행**:
  ```bash
  python tests/test_refresh.py
  ```
- **참고**: 갱신 실패 시 `setup_kakao.bat`으로 재인증 필요

### test.bat — 빠른 테스트 실행 (배치 파일)
- **용도**: 가상환경을 자동 활성화하고 `main.py run --no-deploy` 를 실행합니다.
- **범위**: 전체 파이프라인 (Git 배포 제외)
- **실행**: 더블클릭 또는 터미널에서 `tests\test.bat` 실행
